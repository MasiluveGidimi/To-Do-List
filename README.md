# To-Do List App

## 📌 Project Description

This is a simple **To-Do List Web Application** built with **HTML, CSS, and JavaScript**.
The app allows users to manage daily tasks by adding, completing, deleting, and restoring tasks.

All tasks are saved using **Local Storage**, so they remain available even after refreshing the page.

---

## 🚀 Features

* ➕ Add new tasks
* ✅ Mark tasks as completed (crossed out when checked)
* 🗑️ Delete tasks
* ♻️ Restore the last deleted task
* 📊 Task counter (shows remaining tasks)
* 💾 Save tasks using Local Storage
* 🔄 Tasks remain after refreshing the page

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Local Storage API

## ▶️ How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Double-click **index.html** or open it in a web browser.

---

## 📸 Example Usage

1. Type a task in the input field.
2. Click **Add Task**.
3. Check the checkbox to mark the task as completed.
4. Click **delete** to remove a task.
5. Click **Restore Last Deleted** to bring it back.
